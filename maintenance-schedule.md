# Maintenance Schedule

## Weekly

- verify containers are running
- review free disk space
- quick service access test

## Monthly

- run SMART checks
- apply updates
- verify backups complete
- clean unused Docker images

## Quarterly

- review passwords
- test restore procedure
- inspect fans and dust
- review storage growth

## Yearly

- deeper hardware inspection
- revisit disk upgrade plan
- review network and remote access setup
